### SelectBox

| 属性名 | 说明                                    | 类型    | 默认值 |
| -------- | --------------------------------------- | ------- | ------ |
| mode     | 选择器的展现形式，可选值 box \| button | string  |        |
| vertical | 是否垂直显示                            | boolean | false  |
| checkValueOnOptionsChange | 当选项改变时，检查并清除不在选项中的值  | boolean | false |

更多属性请参考 [Select](/zh/procmp/data-entry/select/#Select)。
