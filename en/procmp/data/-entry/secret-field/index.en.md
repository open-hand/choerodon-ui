Secondary authentication for sensitive information.

## When To Use

When viewing or modifying sensitive information requires secondary authentication.
Before using the desensitization component, configure it in Global Config; see [Demo](/en/procmp/data-entry/secret-field#在表单中显示). Developers do not need additional configuration.

> Added in version 1.5.0.
