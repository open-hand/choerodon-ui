Rate评分

评分组件。

何时使用#

  • 对评价进行展示。

  • 对事物进行快速的评级操作。

代码演示

基本

最简单的用法。

import { Rate } from 'choerodon-ui';

ReactDOM.render(<Rate />, mountNode);
3 stars
文案展现

给评分组件加上文案展示。

import { Rate } from 'choerodon-ui';

class Rater extends React.Component {
  state = {
    value: 3,
  }

  handleChange = (value) => {
    this.setState({ value });
  }

  render() {
    const { value } = this.state;
    return (
      <span>
        <Rate onChange={this.handleChange} value={value} />
        {value && <span className="c7n-rate-text">{value} stars</span>}
      </span>
    );
  }
}

ReactDOM.render(<Rater />, mountNode);
allowClear: true
allowClear: false
清除

支持允许或者禁用清除。

import { Rate } from 'choerodon-ui';

ReactDOM.render(
  <div>
    <Rate defaultValue={3} /> allowClear: true
    <br />
    <Rate allowClear={false} defaultValue={3} /> allowClear: false
  </div>,
  mountNode);
半星

支持选中半星。

import { Rate } from 'choerodon-ui';

ReactDOM.render(<Rate allowHalf defaultValue={2.5} />, mountNode);
只读

只读，无法进行鼠标交互。

import { Rate } from 'choerodon-ui';

ReactDOM.render(<Rate disabled defaultValue={2} />, mountNode);

其他字符

可以将星星替换为其他字符，比如字母，数字，字体图标甚至中文。

import { Rate, Icon } from 'choerodon-ui';

ReactDOM.render(
  <div>
    <Rate character={<Icon type="heart" />} allowHalf />
    <br />
    <Rate character="A" allowHalf style={{ fontSize: 36 }} />
    <br />
    <Rate character="好" allowHalf />
  </div>,
  mountNode);

属性说明类型默认值
